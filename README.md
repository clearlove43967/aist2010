# SuperNoisyMario
A modified supermario game based on https://github.com/marblexu/PythonSuperMario.
* A new map with different new enemies
* Three different game modeï¼š

1) a
2) b
3) c

# Requirement
* Python 3.7
* Python-Pygame 1.9
* Pyaudio 0.2.11

# How To Start Game
$ python main.py

# How to Play
* use A/D key to control player
* use key 'W' to jump
